# SKYNEX Design Tokens

The SKYNEX design system uses a "Tactical Command / Mission Control" aesthetic with deep space colors and neon accents.

## Surface Hierarchy

The "Nesting Depth Principle" — hierarchy is established by shifting background tones rather than drawing boxes.

## Typography

### Font Families

| Role | Font | Purpose |
|------|------|---------|
| Display & Headline | **Space Grotesk** | KPIs and Page Titles |
| Body & Titles | **Inter** | High legibility text |
| Data & Technical | **JetBrains Mono** | IPs, timestamps, logs |

### Type Scale

| Token | Size | Line Height | Weight |
|-------|------|-------------|--------|
| display-lg | 57px | 64px | 700 |
| display-md | 45px | 52px | 700 |
| headline-lg | 32px | 40px | 600 |
| title-lg | 22px | 28px | 600 |
| title-md | 16px | 24px | 600 |
| body-md | 14px | 20px | 400 |
| label-md | 12px | 16px | 500 |

## Spacing Scale (4px base)

| Token | Value |
|-------|-------|
| space-1 | 4px |
| space-2 | 8px |
| space-3 | 12px |
| space-4 | 16px |
| space-6 | 24px |
| space-8 | 32px |

## Corner Radii

| Token | Value | Usage |
|-------|-------|-------|
| radius-xs | 2px | Inline badges |
| radius-sm | 2px | Cards, tight containers |
| radius-md | 6px | Cards, buttons |
| radius-lg | 10px | Modals |
| radius-xl | 12px | Maximum — never exceed |

## Elevation (Shadows)

| Token | Value |
|-------|-------|
| elevation-sm | 0px 2px 8px rgba(0,0,0,0.3) |
| elevation-md | 0px 6px 16px rgba(0,0,0,0.35) |
| elevation-lg | 0px 12px 32px rgba(0,0,0,0.4) |

## Key Rules

1. **No 1px borders for sectioning** — Use tonal shifts instead
2. **Glass effect for floating elements** — 20px backdrop-blur, 60% opacity
3. **Corners never exceed radius-xl (12px)**
4. **JetBrains Mono for any string containing numbers**
