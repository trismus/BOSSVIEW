# Sicherheitsrichtlinie

## Meldung von Sicherheitslücken

Sicherheitslücken **nicht** über öffentliche GitHub Issues melden.

Bitte stattdessen direkt per E-Mail an das Team wenden:

**Kontakt:** security@skynex.local

Folgende Informationen beifügen:
- Beschreibung der Schwachstelle
- Schritte zur Reproduktion
- Mögliche Auswirkungen
- Vorgeschlagene Behebung (falls vorhanden)

Wir bestätigen den Eingang innerhalb von 48 Stunden und informieren über die weiteren Schritte.

## Compliance

SKYNEX wird nach **ISO 27001** entwickelt und betrieben. Dies umfasst:
- Vollständiger Audit-Trail für alle Datenänderungen
- Rollenbasierte Zugriffskontrolle (RBAC)
- Verschlüsselung sensibler Daten
- Lückenlose Change-Dokumentation
- Aufbewahrungsfrist mind. 3 Jahre (Luftfahrt-Industrie)

## Verantwortlichkeiten

Security-Reviews werden von Ioannis (Security Agent) durchgeführt. Bei sicherheitsrelevanten Änderungen ist ein Review durch ihn Pflicht.
